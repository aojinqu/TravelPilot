# MCP AI Travel Planner - React 前端

这是使用 React 构建的旅行规划器前端应用。

## 安装依赖

```bash
npm install
```

或者使用 yarn:

```bash
yarn install
```

## 启动开发服务器

```bash
npm run dev
```

应用将在 http://localhost:3000 上运行。

## 构建生产版本

```bash
npm run build
```

构建的文件将输出到 `dist` 目录。

## 预览生产版本

```bash
npm run preview
```

## 技术栈

- React 18
- Vite
- Axios (HTTP 客户端)
- React Markdown (Markdown 渲染)
- CSS3 (样式)
